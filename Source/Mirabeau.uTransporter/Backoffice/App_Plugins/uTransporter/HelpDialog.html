Help

Sync:

Bring all the document types that are compiled within the .dll to the datebase

Generate:

Generate C# classes from all the document types, tabs and templates. Don't forget to recompile the project before syncing

Remove:

WARNING USE WITH CAUTION.

Removes all the document types, tabs and templates from your Umbraco website

Show Log:

Show the info, debug and error log

Dryrun:

The dryrun action first makes a copy of the database. Second it performs all the transformations on the copy database. The original database will not be modified. Because of this the dryrun action will take long to perform!
